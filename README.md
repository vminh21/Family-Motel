# 🏠 Family Motel Management (Quản Lý Phòng Trọ Gia Đình)

Hệ thống quản lý phòng trọ gia đình tối giản, hiện đại và hiệu quả. Giúp quản lý danh sách phòng, khách thuê, thiết bị phòng và tự động chốt hóa đơn hàng tháng chỉ với 3 bước.

---

## 🛠️ Công Nghệ Sử Dụng (Tech Stack)

* **Frontend**: Next.js 15 (React), Tailwind CSS / Vanilla CSS.
* **Backend**: Node.js, Express.js.
* **Database**: MySQL 8.0 / 8.4.
* **DevOps**: Docker, Docker Compose, Vercel, Render, Aiven.

---

## 🚀 Chạy Nhanh Dự Án Ở Máy Local

Để chạy thử nghiệm toàn bộ hệ thống trên máy cá nhân, đảm bảo máy bạn đã cài đặt **Docker Desktop** và đang chạy:

```bash
# Khởi chạy tất cả các dịch vụ (Frontend, Backend, MySQL)
docker compose up --build -d
```

* **Giao diện Website**: [http://localhost:3001](http://localhost:3001)
* **Backend API**: [http://localhost:5001](http://localhost:5001)
